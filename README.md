# Online Pizza Ordering Website

## Overview

This project is an online pizza ordering website that allows users to browse a menu, customize their pizza orders, and place orders online. The website is built using HTML, CSS, JavaScript, Bootstrap, and PHP.

## Features

- **Browse Menu**: View a variety of pizzas available for order.
- **Customize Orders**: Customize your pizza with different toppings and sizes.
- **Cart Management**: Add items to the cart and view the cart summary.
- **Order Placement**: Place orders and receive confirmation.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## Technologies Used

- **Front-End**: HTML, CSS, JavaScript, Bootstrap
- **Back-End**: PHP
- **Database**: MySQL
